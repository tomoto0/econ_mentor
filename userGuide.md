# EconoMentor - ユーザーガイド

## 概要

**EconoMentor**は、AI経済学メンターと対話を通じて経済学を学習できるプラットフォームです。ユーザー登録不要で、誰でもすぐに経済学の専門的なトピックについて学習できます。

## Powered by Manus

EconoMentorは以下の最先端技術で構築されています：

- **フロントエンド**: React 19 + TypeScript + Tailwind CSS 4
- **バックエンド**: Express 4 + Node.js + tRPC 11
- **データベース**: PostgreSQL
- **AI機能**: Manus LLM API（経済学専門知識を持つAIメンター）
- **グラフ描画**: Chart.js + React-ChartJS-2
- **デプロイ**: オートスケーリングインフラストラクチャとグローバルCDN

## 使い方

### 1. 学習を開始する

ホームページで学習したい経済学のトピックを入力します。例えば：

- 「需要と供給」
- 「インフレーション」
- 「ゲーム理論」
- 「限界効用」
- 「GDP」
- 「金融政策」

「学習を開始」ボタンをクリックすると、新しい学習セッションが開始されます。

### 2. AIメンターと対話する

学習ページで、AIメンターに質問を入力します。以下のようなことができます：

**理論の解説を受ける**
- 「需要と供給の関係について説明してください」
- 「インフレーションの原因は何ですか？」

**グラフを生成する**
- 「需要と供給曲線を描いて」
- 「費用曲線を示してください」
- 「フィリップス曲線を描いて」

**シナリオ分析を実施する**
- 「政府が最低賃金を引き上げたら、市場に何が起こるか？」
- 「金利が上昇したら、経済にどのような影響があるか？」

**最新ニュースとの関連付け**
- 「この理論は最近のニュースとどう関係ある？」

### 3. セッションを復元する

ページを離れて後で戻ってきた場合、前回の学習セッションが自動的に復元されます。チャット履歴も保存されているため、前回の会話から続けることができます。

## 主な機能

### 専門的な解説

AIメンターは経済学の理論を分かりやすく、かつ正確に解説します。数式、定義、実例を含めた包括的な説明を提供します。

### 動的グラフ生成

複雑な経済モデルを視覚化します。以下のグラフに対応しています：

- **需要と供給曲線**: 市場均衡の概念を理解
- **費用曲線**: 平均費用と限界費用の関係
- **フィリップス曲線**: インフレーションと失業率の関係
- **GDP成長**: 経済成長の推移
- **無差別曲線**: 消費者の効用最大化

### リアルタイム対話

質問に即座に応答し、深い理解をサポートします。追加の質問や説明を求めることで、より詳しく学習できます。

### セッション管理

- **複数セッション対応**: 異なるトピックについて複数のセッションを管理可能
- **履歴保存**: すべての対話が保存され、後で復習可能
- **セッション復元**: ブラウザを閉じても、次回訪問時に前回の状態から再開

## レスポンシブデザイン

EconoMentorはデスクトップ、タブレット、スマートフォンのすべてのデバイスで快適に利用できるよう設計されています。

## 次のステップ

EconoMentorを使用して、以下のことを実現できます：

1. **基礎理論の習得**: 経済学の基本概念を深く理解
2. **実践的な応用**: 理論が現実の経済状況にどのように適用されるかを学習
3. **シナリオ分析**: 「もし～だったら」という思考実験を通じて経済的思考を養成
4. **継続的な学習**: セッションを保存して、いつでも学習を再開

Manus AIに「機能の追加」や「改善の提案」をいつでもリクエストしてください。EconoMentorはあなたの学習ニーズに合わせて進化し続けます。
